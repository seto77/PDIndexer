# Crystal Parameter

`Crystal Parameter` は、回折線計算と表示に使う結晶リストを管理するツールです。

## Crystal リスト

チェックされた結晶の回折線が、メイン画面のプロファイル上に表示されます。初期リストの先頭には、EOS 計算で使う標準物質が含まれます。

## 結晶データ

結晶情報エリアで構造データを編集できます。CIF または AMC/AMCSD 形式のファイルからのインポートにも対応します。結晶リストは XML として保存され、関連ツールと共通利用できます。

## 回折ピーク表示

回折ピークをプロファイル上部/下部に表示するか、相対強度を計算するか、近接ピークをまとめるか、指数表示をどの結晶に対して行うかを設定します。
