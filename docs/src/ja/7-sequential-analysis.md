# Sequential Analysis

`Sequential Analysis` は、チェックされた複数プロファイルに対してピークフィッティングと結果出力を連続して実行する機能です。

## 対象

条件を変えながら測定した多数のプロファイルに対して、同じピーク解析手順を繰り返す場合に使います。

## 出力

マクロ API と更新履歴から、2θ、d-spacing、FWHM、ピーク強度、格子定数、圧力、Singh equation の結果出力が確認できます。

## 移行メモ

旧 Word マニュアルでは、この機能の説明はメインウィンドウほど整っていません。現行 GUI、ツールチップ、マクロ API を正として本文化します。
