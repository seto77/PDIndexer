# ファイル形式

PDIndexer は、プロファイルファイル、結晶リストファイル、結晶構造インポートファイルを扱います。

## プロファイルデータ

既知のプロファイル関連形式には `pdi`, `pdi2`, `csv`, `tsv`, `ras`, `nxs`, `npd`, `chi`, `xbm`, `rpt` などがあります。詳細な挙動は、本文移行時に現行の読み込みコードと照合します。

## 結晶データ

結晶リストは XML として保存されます。結晶構造は CIF および AMC/AMCSD 形式からインポートできます。

## エクスポート

プロファイルデータは CSV として出力できます。描画中のプロファイルは、PowerPoint や Word で使いやすい EMF メタファイルとして保存できます。
