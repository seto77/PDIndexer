# トラブルシューティング

このページには、インストール、読み込み、表示、解析で起こりやすい問題をまとめます。

## 起動しない

インストール後に PDIndexer が起動しない場合は、現在のアプリに対応する .NET Desktop Runtime が入っているか確認してください。

## クリップボード転送

IPAnalyzer からのクリップボード転送がうまく動かない場合は、`Option` メニューの `Watch Clipboard` をいったんオフにしてから、再度オンにしてください。

## プロファイル読み込み

テキスト形式のプロファイルが読み込めない場合は、区切り文字、小数点記号、列の順序、ヘッダー行の有無を確認してください。

## 移行メモ

トラブルシューティング項目は、更新履歴、Issue、よくある問い合わせをもとに拡充します。
