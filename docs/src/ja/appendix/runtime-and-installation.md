# 動作環境とインストール

PDIndexer は Windows 上で動作し、現在は .NET Desktop Runtime 10.0 を対象にしています。

## インストール

GitHub の最新 Release から `PDIndexerSetup.msi` をダウンロードし、インストーラを実行します。

## ランタイム

必要なのは通常の .NET Runtime ではなく、Windows デスクトップアプリ用の .NET Desktop Runtime です。ランタイムのリンクは `README.md` の記述を更新時に確認します。
