# Profile Parameter

`Profile Parameter` は、読み込まれたプロファイルの表示・補正設定を編集するツールです。

## Profile リスト

メイン画面の Profile リストと対応し、選択、順序、表示状態、削除、描画色などを管理します。

## Smoothing

選択プロファイルに Savitzky-Golay 型の局所フィットによる平滑化を行います。パラメータ説明は、旧マニュアルから移行したうえで現在の GUI と照合します。

## Back Ground

バックグラウンド減算を自動計算できます。自動曲線が合わない場合は、Edit mode でプロファイル上の制御点を手動調整します。

## 関連操作

現行アプリには、軸補正、プロファイル演算、K-alpha2 除去などの処理もあります。本文移行の進行に合わせて節を分けます。
