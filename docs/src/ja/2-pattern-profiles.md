# プロファイルデータ

プロファイルデータは、角度-強度または変換後の横軸-強度として PDIndexer に読み込まれるデータです。

## 読み込み

プロファイルは `File` メニュー、ドラッグ&ドロップ、IPAnalyzer からのクリップボード転送で読み込めます。専用形式に加えて、角度-強度の2列テキストも読み込み対象です。

## 表示モード

単一プロファイルモードと複数プロファイルモードを切り替えられます。複数プロファイルモードでは、読み込まれたプロファイルをリストに保持し、チェックされたものを重ねて表示します。

## 表示・補正

スムージング、バックグラウンド減算、色指定、複数プロファイル間の強度オフセット、横軸変換などを扱います。

## 表記方針

表示変換は、元データそのものの変更とは限りません。コントロール名を書くときは、旧マニュアルの表現よりも現在の `.resx` の GUI ラベルを優先します。
