# メインウィンドウ

メインウィンドウは、プロファイルの読み込み、結晶の選択、横軸表示の変更、解析ツールの起動を行う中心画面です。

## File メニュー

`File` メニューでは、プロファイルの読み書き、結晶リストの読み書き、CIF/AMC のインポート、印刷、メタファイル保存、終了を扱います。

## Option / Help メニュー

`Option` メニューでは、ツールチップ表示やクリップボード監視などの動作を切り替えます。`Help` メニューでは、バージョン情報、ヒント、マニュアルを開きます。

## Horizontal Axis

横軸表示の設定を変更します。代表的な表示モードは `2θ`、X線散乱角、電子線散乱角、エネルギー、d-spacing です。

## プロファイル表示領域

中央の表示領域には、測定プロファイルと計算された回折線を表示します。マウス操作で拡大縮小、表示範囲の移動、選択結晶の回折線移動を行います。

## Profile / Crystal リスト

Profile リストでは表示するプロファイルを、Crystal リストでは重ねて表示する結晶の回折線を選びます。
