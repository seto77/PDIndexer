# Fitting diffraction peaks

`Fitting diffraction peaks` は、観測ピークを関数フィッティングし、選択した回折線から格子定数を精密化するツールです。

## 基本手順

対象となる結晶とプロファイルを選択し、計算回折線を観測ピークに近づけ、フィッティングする反射を選び、精密化された格子定数を選択結晶へ反映します。

## Fitting Option

旧マニュアルには、Simple Search、Symmetric Pseudo Voigt、Symmetric Pearson VII、Split Pseudo Voigt、Split Pearson VII、Search Range、Pattern Decomposition が挙げられています。最終本文では、現行 GUI の表示名に合わせます。

## 出力

フィッティング結果は、表計算ソフトへ貼り付けやすい形式でコピーできます。Sequential Analysis は、この作業を複数プロファイルに展開する機能です。
