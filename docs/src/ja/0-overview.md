# 概要

PDIndexer は、X線、放射光X線、中性子TOFなどの粉末回折プロファイルを扱うための解析ソフトです。プロファイルの表示・補正、結晶データに基づく回折線表示、ピーク解析、圧力計算を主な対象にしています。

## 主な機能

- `pdi`, `pdi2`, `csv`, `tsv`, `ras`, `nxs`, `npd`, `chi`, `xbm`, `rpt` などのプロファイルを読み込みます。
- IPAnalyzer からクリップボード経由でプロファイルを受け取れます。
- 単一/複数プロファイルを表示し、横軸の表示単位を切り替えられます。
- CIF および AMC/AMCSD 形式の結晶構造データを読み込めます。
- 回折ピーク位置と相対強度を計算します。
- 回折ピークをフィッティングし、格子定数を精密化します。
- 標準物質の EOS を使って圧力を計算します。
