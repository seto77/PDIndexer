# Crystal Parameter

The `Crystal Parameter` tool manages the crystal list used to calculate and display diffraction lines.

## Crystal List

Checked crystals are drawn on the main profile view. The first bundled entries include standard materials used by the EOS tool.

## Crystal Data

Crystal structures can be edited in the crystal information area and imported from CIF or AMC/AMCSD-style files. Crystal lists are saved as XML and are compatible with related SetoLab tools.

## Diffraction Peak Options

Diffraction peak display options control whether peaks appear above or below the profile, whether relative intensities are calculated, how adjacent peaks are combined, and which peak indices are labeled.
