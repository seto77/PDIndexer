# Sequential Analysis

`Sequential Analysis` applies peak fitting and result export workflows to a set of checked profiles.

## Scope

This feature is useful when many profiles are measured under a changing condition and the same peak-analysis procedure should be repeated across the series.

## Results

The macro API and version history indicate outputs such as 2θ, d-spacing, FWHM, peak intensity, cell constants, pressure, and Singh-equation results.
