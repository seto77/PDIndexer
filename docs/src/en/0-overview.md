# Overview

PDIndexer is designed for powder diffraction studies using X-ray, synchrotron X-ray, neutron TOF, and related profile data. It focuses on practical profile inspection, diffraction-line comparison, and peak analysis.

## Main Features

- Reads common profile formats such as `pdi`, `pdi2`, `csv`, `tsv`, `ras`, `nxs`, `npd`, `chi`, `xbm`, and `rpt`.
- Receives profiles from IPAnalyzer through the clipboard.
- Displays single or multiple profiles with configurable horizontal axes.
- Imports crystal structures from CIF and AMC/AMCSD-style data.
- Calculates diffraction peak positions and relative intensities.
- Fits diffraction peaks and refines lattice constants.
- Calculates pressure with built-in EOS entries for standard materials.
