# Fitting Diffraction Peaks

The `Fitting diffraction peaks` tool fits observed diffraction peaks and refines lattice constants from selected reflections.

## Basic Flow

Select the target crystal and profile, move calculated diffraction lines close to observed peaks, choose the reflections to fit, and apply the refined lattice constants back to the selected crystal.

## Fitting Options

Legacy documentation lists simple search, symmetric pseudo-Voigt, symmetric Pearson VII, split pseudo-Voigt, split Pearson VII, search range, and pattern decomposition. The current GUI should be used to confirm exact option names.

## Output

Fitting results can be copied in a spreadsheet-friendly format. Sequential analysis builds on this workflow for multiple profiles.
