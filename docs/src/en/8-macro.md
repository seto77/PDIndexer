# Macro

PDIndexer exposes a macro API for automating common profile, crystal, fitting, and sequential-analysis operations.

## Sources

Macro documentation should be generated or curated from `PDIndexer/Macro.cs`, especially the `Help` attributes attached to public macro members.

## Topics to Cover

- Current view and axis operations.
- Crystal list selection, checking, and cell-parameter access.
- Profile list selection, checking, and arithmetic operations.
- Fitting window commands and fitting-parameter access.
- Sequential analysis execution and auto-save settings.

## Editing Rule

When macro names and GUI names differ, keep the macro API spelling for code examples and use current GUI labels for UI steps.
