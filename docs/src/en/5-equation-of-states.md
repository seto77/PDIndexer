# Equation of States

The `Equation of States` tool calculates pressure from selected standard materials and their current lattice parameters.

## Linked Crystals

The EOS calculation uses standard entries from the crystal list. Legacy documentation mentions Au, Pt, NaCl B1, NaCl B2, MgO, and Al2O3; the current bundled database and GUI should be treated as authoritative.

## Workflow

Select a material, adjust or fit its lattice parameter, then read the pressure result in the EOS window. Changes made by dragging diffraction lines in the main window can update the calculation immediately.
