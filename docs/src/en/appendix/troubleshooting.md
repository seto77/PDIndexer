# Troubleshooting

This page collects problems that are likely to appear during installation, loading, display, and analysis.

## Runtime Problems

If PDIndexer does not start after installation, confirm that the .NET Desktop Runtime matching the current application version is installed.

## Clipboard Transfer

If clipboard transfer from IPAnalyzer does not work, turn `Watch Clipboard` off and on again from the `Option` menu.

## Profile Loading

When a text profile fails to load, confirm the delimiter, decimal separator, column order, and whether the file contains non-data header lines.
