# Runtime and Installation

PDIndexer runs on Windows and currently targets .NET Desktop Runtime 10.0.

## Install

Download `PDIndexerSetup.msi` from the latest GitHub release and run the installer.

## Runtime

Install the .NET Desktop Runtime, not only the base .NET Runtime. The runtime link is maintained in the repository `README.md` and should be checked when this page is updated.
