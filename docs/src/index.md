# PDIndexer Manual

PDIndexer is a Windows application for powder diffraction profile analysis. It reads diffraction profiles, displays and processes them, overlays calculated diffraction lines from crystal structures, fits peak positions, and supports pressure estimation with equations of state.

!!! note "Migration status"
    This GitHub Pages manual is the new editable documentation tree. The first pass establishes the folder structure, navigation, and starter Markdown pages. Detailed text and screenshots will be migrated from the legacy files under `PDIndexer/doc/`.

Start with [Overview](en/0-overview.md), then move through [Main window](en/1-main-window.md), [Pattern profiles](en/2-pattern-profiles.md), and the analysis tools.
